#  Irrigation Need Prediction Project

##  Overview
This project aims to predict the irrigation needs of crops based on environmental and soil features such as:
- Sunlight hours
- Humidity
- Soil organic carbon
- Soil PH

##  Dataset
The dataset includes features related to:
- Weather conditions  
- Soil properties  
- Crop requirements  

##  Methodology
The project follows these steps:
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Model Training  
5. Model Evaluation
